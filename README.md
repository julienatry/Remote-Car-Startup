# Remote-Car-Startup

Make your car start by pressing a button on your phone


# Components used
- Programmable µC (ATmega 328P)
- Relays (SRD-05VDC-SL-C)
- Bluetooth module (HC-05)
- NPN Transistors (S8050)
- Diode (IN4007)
- 16MHz Quartz Crystal
